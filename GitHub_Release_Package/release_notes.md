# v1.5.0 更新说明

感谢您使用 CANlearn Pro！本次更新带来了以下改进：

## ✨ 新增功能 (New Features)

- [x] **协议支持**: 添加 CRC-16/ModBus-LE 小端序支持
- [x] **快捷操作**: 清屏添加快捷键 (Ctrl+L)
- [x] **数据导出**: 支持另存为字符串、十六进制、二进制格式
- [x] **编码支持**: 添加全部文本编码支持
- [x] **日志功能**: 添加日志可编辑功能

## 🚀 优化与修复 (Improvements & Fixes)

- [x] 修复 MessageCard 组件报错崩溃问题
- [x] 优化高负载下的 UI 响应速度
- [x] 修复特定场景下的内存泄漏

---

### 如何安装
下载下方的 `CANlearn_Setup.exe` 或 `.zip` 解压运行即可。
